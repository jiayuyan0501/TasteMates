# TasteMates

A full-stack mobile application for food lovers to discover and share their favorite meals.

## Project Structure

```
TasteMates/
├── frontend/          # React Native mobile app
│   ├── App.js
│   ├── package.json
│   └── assets/
├── backend/           # Node.js/Express API server
│   ├── index.js
│   ├── package.json
│   ├── routes/
│   ├── controllers/
│   ├── models/
│   └── middleware/
└── README.md
```

## Getting Started

### Frontend (React Native)

```bash
cd frontend
npm start
```

### Backend (Node.js/Express)

```bash
cd backend
npm start
```

## Development

- Frontend runs on Expo development server
- Backend runs on port 3001
- Use Expo Go app to test on mobile devices
